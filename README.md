# mainweb-i-hub

## 项目简介

`mainweb-i-hub` 是一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为多个独立 Web 页面的集合与分发点。每个页面均为自包含的 HTML 文件，可直接在浏览器中打开查看。

## 目录结构

```
mainweb-i-hub/
├── pages/           # 存放所有独立 HTML 页面
├── assets/          # 公共资源文件（CSS、JS、图片等）
├── index.html       # 仓库首页（可选）
└── README.md        # 本文件
```

- `pages/`：每个子目录或文件代表一个独立页面，文件名即页面标识。
- `assets/`：存放页面间共享的静态资源，按需引用。
- `index.html`：可选的主页，用于展示页面列表或导航。

## 页面归档说明

- 所有页面均为标准 HTML 格式，无外部依赖或跟踪代码。
- 页面内容保持原始形态，不进行修改或二次加工。
- 归档范围包括但不限于：信息页、工具页、测试页、存档页等。
- 每个页面自带样式与脚本，确保离线或在线均可正常使用。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 下的任意 HTML 文件。
3. 也可通过 GitHub Pages 等服务部署后在线访问。

## 维护说明

- 本仓库由维护者不定期更新，新增或更新页面会通过 commit 记录。
- 不接收外部提交的页面，仅由维护者自行归档。
- 如发现页面显示异常或链接失效，可提交 Issue 反馈。
- 仓库不提供任何形式的保证或支持，仅作存档用途。
